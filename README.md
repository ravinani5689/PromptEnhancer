# ⚡ PromptForge — AI Prompt Optimizer & Sandbox

**PromptForge** is a lightweight, single-page web application that converts rough prompt ideas into structured, production-grade AI prompts. It includes a built-in sandbox to test and execute your refined prompts in real time.

---

## ✨ Features

* **Meta-Prompt Engineering:** Automatically restructures rough thoughts into clean frameworks featuring `Role`, `Context`, `Task`, `Constraints`, and `Output Format`.
* **Real-time Execution Sandbox:** Test your engineered prompts immediately and view model outputs.
* **Multi-Provider Support:** Works with **Groq Cloud** (Llama 3.3 70B) or **OpenAI** (GPT-4o / GPT-4o-mini).
* **Zero Dependencies:** Pure HTML, Tailwind CSS, and Vanilla JavaScript—no build steps or framework setup required.
* **Privacy Focused:** Your API keys are stored locally in your browser (`localStorage`) and never sent to any middleman server.

## ⚙️ How to Setup & Run

### **Option 1: Use the Live Site**
1. Visit the [Live Link](https://ravinani5689.github.io/PromptEnhancer/).
2. Click **⚙️ Settings & API Key** at the top right.
3. Paste a free API key from [GroqCloud Console](https://console.groq.com) (or OpenAI) and click **Save Settings**.
4. Enter any rough prompt idea and click **✨ Engineer & Optimize**.
